# Almacenamiento de remedios

Primera version de inventario domestico para medicamentos. Funciona como app web estatica y guarda los datos en el navegador del dispositivo.

## Uso

1. Abre `index.html` en un navegador.
2. Agrega medicamentos con nombre, cantidad, vencimiento y ubicacion.
3. Usa busqueda y filtro por estado para encontrar rapido lo que hay en casa.
4. Exporta o importa el inventario con los botones de flecha en la esquina superior.

## Siguiente mejora sugerida

Para ver los mismos datos en varios dispositivos sin copiar archivos, el siguiente paso es agregar sincronizacion. Las opciones mas simples son una base de datos local en red, Firebase/Supabase o un pequeno backend propio.
